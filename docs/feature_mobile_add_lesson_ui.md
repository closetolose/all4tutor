# Фича: Улучшение UI формы создания урока (мобильная версия)

## 1. Требования и цели

**Цель:** Сделать форму создания урока на мобильной версии более красивой, эргономичной и удобной. Все элементы должны стоять ровно и предсказуемо.

**Область изменений:**
- Файл: `core/templates/mobile/add_lesson.html`
- Только визуал и вёрстка (CSS, при необходимости обёртки). Не менять: `id`, `name`, `action`, `method`, скрытые поля, классы для JS.

**Ограничения:**
- Бэкенд и логика формы не трогаем.
- Сохраняем touch-цели ≥ 44px и доступность.

---

## 2. User Stories и критерии приёмки

| # | User Story | Критерии приёмки |
|---|------------|------------------|
| 1 | Как пользователь, я хочу, чтобы все подписи и иконки в строке были выровнены по одной линии | Иконки в `.tg-label` и текст на одной вертикальной оси; стрелка `.tg-arrow` по центру строки |
| 2 | Как пользователь, я хочу, чтобы подписи не переносились на две строки | «Длительность», «Место / Ссылка» и др. в одну строку (`white-space: nowrap`, при необходимости `min-width`) |
| 3 | Как пользователь, я хочу удобно нажимать на все кнопки и поля с телефона | Кнопки и кликабельные зоны не менее 44×44px; адекватные отступы между блоками |
| 4 | Как пользователь, я хочу видеть аккуратную визуальную иерархию | Карточки, заголовки секций и строки полей с единой сеткой отступов и скруглений |

---

## 3. Технический подход

- **Сетка:** Flexbox для всех строк (`.tg-input-row`, `.lesson-card-title`, `.segment-option`). Единый `gap` и `padding` по карточкам.
- **Выравнивание:** `align-items: center` для строк; иконки в подписях — `inline-flex` + `align-items: center` + один размер шрифта с текстом.
- **Типографика:** Общие переменные `--tg-*` и при необходимости `--font-*` из дизайн-системы; единый `line-height` для строк.
- **Контур и размеры:** Кнопка «Прикрепить файлы» уже уменьшена; при необходимости ещё подправить отступы и радиус.
- **Стили:** Все правки в блоке `<style>` внутри `add_lesson.html` (мобильная страница отдельная, десктоп не затрагивается).

---

## 4. Архитектура (UI)

- **Структура:** Заголовок страницы → сегмент «Индивидуально/Группа» → карточки секций (Кто и что, Время и стоимость, Формат и место, Повторение, Материалы) → кнопка «Создать занятие».
- **Компоненты:** `.lesson-form-wrap`, `.lesson-card`, `.lesson-card-title`, `.tg-input-row`, `.tg-label`, `.tg-field-wrap`, `.tg-arrow`, `.mobile-fp-trigger-addlesson`, `.mobile-submit-btn`. Без новых зависимостей.
- **Тестирование:** Ручная проверка на ширине 320–428px; проверка, что все поля и кнопки работают (отправка формы, выбор ученика/группы, переключатель, файлы).
