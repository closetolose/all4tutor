# Задача: Этап 4 — Теги файлов

**План:** `docs/plan.md`, этап 4 (комплексное обновление).

## Чек-лист

- [ ] **4.1** Модель `FileTag`: поля `tutor` (FK → Users), `name` (CharField). Meta: unique_together (tutor, name). Миграция.
- [ ] **4.2** В модель `FilesLibrary` добавить M2M: `tags = ManyToManyField(FileTag, blank=True, related_name='files')`. Миграция.
- [ ] **4.3** Зарегистрировать `FileTag` в Django Admin; в админке `FilesLibrary` отобразить теги (list_display / inline при необходимости).
- [ ] **4.4** При загрузке файла (форма в files_library): возможность выбора/создания тегов (репетитор). Создание тега: проверка прав (только свои), уникальность имени у репетитора.
- [ ] **4.5** При редактировании файла (edit_file): возможность добавить/убрать теги.
- [ ] **4.6** В шаблоне files_library: блок фильтра по тегам (множественный выбор). Запрос: при выбранных тегах — фильтр `tags__in=...`; совмещать с поиском по названию (q). При пустом выборе тегов — показывать все файлы.
