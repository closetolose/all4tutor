# Задача: Этап 3 — UI Настройки (форма выбора цвета)

**План:** `docs/plan.md`, этап 3.

## Чек-лист

- [ ] **3.1** В шаблонах `tutor_card` (core и mobile): блок «Цветовая метка» — палитра 6–8 цветов и/или `<input type="color">`, кнопки «Сохранить» и «Убрать цвет». Форма POST на `update_tutor_color` (connection_id из связи student–tutor).
- [ ] **3.2** В шаблонах `my_tutors` (core и mobile): для каждой связи кнопка/иконка «Настроить цвет» → модалка или inline с формой выбора цвета, сохранение на `update_tutor_color`.
- [ ] **3.3** В шаблонах `group_card` и `edit_group` (core и mobile): блок «Цвет группы» — палитра + input color + «Убрать цвет», POST на `update_group_color(group_id)». Проверка доступа (репетитор/ученик).
- [ ] **3.4** Задать предзаданную палитру 6–8 hex-цветов (безопасных для светлой и тёмной темы), без внешних библиотек.
